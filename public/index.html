在 Mac 下使用 Hugo 写 Blog 的安装与部署

本文介绍如何在 Mac 上安装 Hugo、编写博客并部署到 GitHub Pages，整个过程简单易懂。 一. 环境准备与安装 A. 安装 Hugo brew install hugo B. 创建站点 新建 Hugo 站点（使用 YAML 格式配置） hugo new site wctd_blog --format yaml 初始化 Git 仓库 git init C. 配置主题 添加 PaperMod 主题 git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/hugo-PaperMod 二. 编写 blog 与预览 A. 创建要写的文章 hugo new posts/first-post.md B. 编辑博客内容 使用编辑器（vim 或 VSCode）编辑文章 vim posts/first-post.md C. 本地预览 启动本地服务器查看效果 hugo server -D 三. 部署与发布 A. GitHub 仓库配置 添加远程仓库 git remote add origin https://github.com/oooshuang/blog.git B. GitHub Pages 设置 登录 GitHub，进入 blog 仓库 进入 Settings → Pages 页面 选择 Source 为 GitHub Actions 配置 Hugo 工作流，确保 HUGO_VERSION 与本地版本一致 C. 发布与访问 推送到 GitHub 仓库 git push -u origin main 访问博客 完成部署后，通过以下链接访问： https://oooshuang.github.io/blog/posts/first-post/
